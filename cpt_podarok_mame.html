Амбициозная задача¶

Для выполнения задания я выбрал гамма-распределение. Одно из самых самодостаточных и уверенных в себе распределений. Согласно Википедии его математическое ожидание равняется $k\theta$ , а дисперсия $k\theta^2$ . По умолчанию, параметр $\theta$ (иначе - scale) равен 1.

In [216]:
# Задаем параметр shape (k) и строим гамма-распределение. Делаем выборку объёма 1000.

shape = 2

gamma_dis = sts.gamma(shape)
sample = gamma_dis.rvs(1000)

# Строим гистограмму выборки и график теоретической плотности распределения вероятности.

x = np.linspace(gamma_dis.ppf(0.01), gamma_dis.ppf(0.99), 100)
plt.hist(sample, bins = 30, normed=True)
plt.ylabel('$f(x)$')
plt.xlabel('$x$')
pdf = gamma_dis.pdf(x)
plt.plot(x, pdf, lw = 4, color = 'y')

# При выборке n_1 = 5, вычисляем значение дисперсии

n_1 = 5
disp1 = float(shape)/n_1 

# Нормальное распределение

m = shape
std1 = (disp1**(0.5))
norm_rv = sts.norm(m, std1)


# Нам нужна 1000 выборок из гамма-распределения, приступим !


sample_rv1 = [np.mean(gamma_dis.rvs(n_1)) for i in range(1000)]

x = np.linspace(gamma_dis.ppf(0.01), gamma_dis.ppf(0.99), 100)

# Гистограмма


plt.hist(sample_rv1, bins = 30, normed=True, color = 'r')

plt.ylabel('$f(x)$')
plt.xlabel('$x$')

# Кроме этого, нанесём плотность нормального распределения

pdf = norm_rv.pdf(x)
plt.plot(x, pdf, lw = 4, label='theoretical pdf', color = 'b', alpha=0.5)
plt.legend()
